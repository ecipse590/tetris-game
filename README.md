# Tetris Web Game

A classic Tetris game built using HTML, CSS, and JavaScript.
Includes an online leaderboard powered by Firebase.

## Features
- Classic Tetris gameplay
- Scoring, levels, and line clearing
- Next-piece preview
- Online leaderboard

- HTML
